# Webly SV

Sitio web corporativo de Webly SV, microempresa salvadoreña dedicada al desarrollo de sitios web para emprendedores y pequeñas empresas. Este proyecto representa nuestra presencia digital: quien somos, que ofrecemos y como contactarnos.

## Estructura del repositorio

```
weblysv/
├── assets/
│   ├── img/
│   ├── videos/
│   └── favicon.ico
├── css/
│   └── styles.css
├── js/
│   └── scripts.js
├── index.html
├── sobrenosotros.html
├── portafolio.html
├── servicios.html
├── contactos.html
└── forms.html
```

## Paginas

- **index.html** - Inicio: vision, mision y valores
- **sobrenosotros.html** - Presentacion del equipo
- **portafolio.html** - Proyectos realizados
- **servicios.html** - Paquetes y precios
- **contactos.html** - Formulario de contacto
- **forms.html** - Formulario de compra

## Tecnologias

- HTML5, CSS3, JavaScript
- Bootstrap 5
- Font Awesome 6
- Google Fonts (Montserrat, Roboto Slab)
- React, Next.js, Vue.js (proyectos del portafolio)
- Node.js, Firebase (proyectos del portafolio)
- WordPress + WooCommerce (proyectos del portafolio)
- Leaflet.js, Tailwind CSS (proyectos del portafolio)
